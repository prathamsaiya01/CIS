# 💀 Crime Investigation System (CIS)

An interactive, logic-based crime simulation game built using Python (Tkinter), where users can experience both sides of crime:

- 🕵️ Investigator Mode – Solve cases using logic, clues, and deduction  
- 🕶️ Criminal Mode – Execute missions strategically without getting caught  

---

## 🎮 Game Overview

CIS is designed to simulate real-world crime scenarios where players must:

- Analyze situations  
- Make decisions under pressure  
- Solve logical puzzles  
- Experience multiple endings based on choices  

---

## 🚀 Features

- 🎭 Dual Gameplay Modes (Investigator vs Criminal)
- 🧠 Logical + Real-world puzzles (not just ciphers)
- 🔊 Sound effects (correct/wrong feedback)
- 🎯 Decision-based outcomes
- 💻 Terminal-style UI (Tkinter)
- 🗂️ SQLite database (login/signup system)
- 🔥 Multiple levels & cases

---

## 🕵️ Investigator Mode

Solve cases like:
- Murder  
- Theft  
- Cyber Crime  
- Kidnapping  

### Gameplay:
- Crime scene analysis  
- Evidence collection  
- Interrogation  
- Deduction-based puzzles  

### Endings:
- ✅ Case Solved  
- ⚠️ Partial Solve  
- ❌ Wrong Conclusion  

---

## 🕶️ Criminal Mode

Execute missions like:
- Street Theft  
- Bank Heist  
- Perfect Crime  
- Cyber Attack  

### Gameplay:
- Strategic decisions  
- Risk vs reward system  
- Hacking puzzles  
- Escape sequences  

### Endings:
- 😈 Perfect Crime  
- ⚠️ Identified  
- 💀 Caught
